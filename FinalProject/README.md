# VectorFlow

VectorFlow 是一个面向计算机图形学课程设计的矢量流程图编辑系统。它不只是能画流程图，还做了不少更像完整绘图软件的功能：画好的图形可以丢进物理沙盒里碰撞，电路图可以一键通电演示，SQL 可以一键生成 ER 图，常用图形组合可以存进自己的组件库，图片也可以导入到画布里继续排版，并支持 PNG 与 SVG 导出。

## 核心创新点

1. **矢量物理沙盒**
   这是项目里最突出的功能。画完图以后点「▶ 播放」，画布上的图形会变成刚体：圆形会往下掉，矩形和多边形会互相碰撞，直线和手绘曲线可以当斜坡或障碍物。这个效果不是接第三方物理库做的，而是自己写了一个 2D 刚体引擎，里面包括凸包、SAT 碰撞检测、冲量响应、摩擦、角动量和位置修正。点停止后图形会回到原来的位置，所以它更像一个附加的演示模式，不会把原来的作品弄乱。

2. **一键通电电路秀**
   内置了一个更完整的双分支电路模板，不只是电池接一个 LED。点击“电路秀”会生成电池、开关、电阻、电容、电感、两个 LED、接地和电压源组成的电路；点击“通电”后，程序会按当前画布上的真实元件和连线判断哪些路径是通的，再让电流沿对应线路流动、让对应 LED 发光。断开开关后电流停止；删掉 R1 这类主干元件会整条电路断电；删掉 R2 或某条分支线时，只会影响对应分支。这个功能适合短时间演示，因为观众能直接看到“编辑电路 -> 通电反馈”的变化。

3. **SQL 一键生成 ER 图**
   工具栏里有 `SQL -> ER`。把 `CREATE TABLE` SQL 粘进去，程序会解析表名、字段、主键和外键，然后自动生成 ER 图。外键关系会变成表之间的连线，字段里会标出 PK / FK。里面还放了电商订单、博客内容、学校选课几个模板，答辩时可以直接加载模板演示。

4. **自己的组件库**
   可以选中一组图形先组合成一个整体，也可以保存到“我的组件”里。保存时会把组内图形、内部连接线和 Metadata 一起存下来，下次点击组件名就能重新放回画布，而且会自动生成新的 id，不会和原来的图形冲突。普通组合支持取消组合，适合临时整理版面；组件库适合保存常用结构，比如一组电路元件、一个服务节点、一个固定的流程片段。

5. **图片导入**
   支持把 PNG / JPG / BMP / GIF / WebP 等图片导入画布。导入后图片不是临时贴在界面上，而是作为 `RasterImageShape` 存在，可以选中、移动、缩放、复制、删除，也能跟 `.vflow` 文件一起保存，再打开时还在。大图导入时会按显示尺寸压缩，避免文件体积一下变得太大。

6. **算法回放**
   选中一个图形后可以播放它的生成过程。比如直线会按 Bresenham 算法一点一点画出来，圆和椭圆会按中点算法逐步出现，填充区域会展示扫描线填充的过程。这样答辩时不只是说“我实现了算法”，而是能直接把算法怎么工作的过程放出来看。

7. **PNG 像素导出与 SVG 真矢量导出**
   导出 PNG 时，核心几何图形不是调用 `ImageDraw.line`、`ImageDraw.rectangle`、`ImageDraw.ellipse` 这类现成绘图函数画出来的，而是先用自己写的算法算出像素点，再写入 Pillow 图像缓冲区。SVG 导出则会把流程图图形、线条、文本、连接线和 Bezier 曲线写成真正的 SVG 元素，导入的位图图片用 `<image>` 嵌入，方便继续放大或放到矢量软件里处理。

8. **会动的连接线**
   普通流程图的线是静态的，这里可以让连接线上的高亮点沿路径流动，用来表示流程方向，画电路图时也可以表示电流方向。这个动画是沿着连接线采样出来的像素路径做相位偏移，不是简单地贴一个装饰效果。

9. **图元类型比较全，曲线也能继续编辑**
   编辑器里不只有基础几何图形，还放了流程图、组织结构图和电路图符号。手绘曲线不是画完就变成一张死图，而是会经过 Catmull-Rom / Bezier 平滑处理，之后还能选中、移动、变换和撤销。Bezier 工具支持像 Lab4 一样持续添加任意数量控制点、拖拽控制点实时调整曲线、右键删除控制点，再按 Enter 或按钮完成曲线。

## 基础功能

这些功能对应作业对“具有编辑能力的矢量图形编辑系统”的基础要求。

- **流程图图元**：处理框、判断框、起止框、数据框、文档框、数据库、子程序框
- **通用图形**：圆形、椭圆、三角形、梯形、平行四边形、圆角矩形、五角星、六边形、左/右箭头、加号
- **电路图符号**：电阻、电容、接地、电池、开关、LED、电感、电压源（仅绘制器件轮廓，无外侧包围框）
- **一键通电电路秀**：生成双分支电路模板，支持通电流动、LED 发光、开关断开、制造故障，并会根据当前画布上实际存在的元件和连线重新判断通路
- **连接线**：直线 / 正交折线 / 贝塞尔曲线三种线型，拖拽方式创建，支持实线 / 虚线 / 点线
- **箭头样式**：实心箭头、空心箭头、菱形、圆点，起点终点可独立设置
- **直线与文本**：独立直线绘制、内联文本编辑（支持多行、字号、居中/左对齐/右对齐、加粗、颜色）
- **手绘曲线**：按住鼠标自由勾画，采样点经 Catmull-Rom → 三次 Bezier 平滑后写入画布，可被选中、变换、撤销
- **Bezier 曲线**：左键不断添加控制点，点中已有控制点可拖拽调整，右键删除控制点，Backspace 删除最后一点，Enter 或右侧按钮完成曲线；控制点数量不限，使用 De Casteljau 算法采样生成任意阶 Bezier 曲线
- **橡皮擦**：提供「整体擦除」和「部分擦除」两种模式。整体擦除拖拽经过即移除整个对象（图形 / 画笔曲线 / 连接线）；部分擦除仅断开擦除圆经过的笔画片段、保留其余部分（作用于画笔曲线与直线），判定半径可调，擦除后可撤销
- **拖拽确定大小引入图元**：从图形库点选图元后，在画布上按住拖拽即可实时预览并按矩形大小创建，单击则以默认尺寸放置，无需放置后再调整
- **组合与取消组合**：多选图形后可用 Ctrl+G 或右侧属性 / 右键菜单组合，组内图形和内部连接线会成为一个整体一起移动、缩放、旋转、复制、保存和导出；选中组合后可用 Ctrl+Shift+G 取消组合，恢复组内图形和内部连接线
- **我的组件库**：把当前选择保存为自定义组件，保留内部图形、连接线和 Metadata，可再次插入或删除
- **SQL -> ER 图**：解析 `CREATE TABLE` SQL，自动生成带 PK / FK 标记和外键连线的 ER 图，内置多个演示模板
- **图片导入**：支持导入常见位图格式，图片可移动、缩放、复制、删除，并随 `.vflow` 文件保存
- **编辑操作**：选择、框选、移动、复制粘贴、删除、组合 / 取消组合、锁定 / 解锁、橡皮擦（整体/部分），连接线自动跟随
- **图层面板**：左侧常驻图层列表，每个对象占一行（顶行为最上层），支持显示 / 隐藏、锁定、双击重命名、▲ / ▼ 调整层级、单击联动选中；隐藏对象点击会穿透到下层，锁定对象不可拖拽，锁定 / 解锁也可用 Ctrl+L / Ctrl+Shift+L；「层级」菜单与右键菜单提供置顶 / 置底 / 上移 / 下移，并配 Ctrl+↑/↓、Ctrl+Shift+↑/↓ 快捷键；显隐 / 锁定 / 重命名 / 调序均可撤销，并随 `.vflow` 文件保存
- **界面布局**：顶部菜单栏收纳新建 / 打开 / 保存 / 导入图片 / 导出 PNG / 导出 SVG 等文件操作；命令栏只保留 ER 图、算法回放、电路演示、主题切换等常用功能，并随窗口宽度自动换行、不再左右滚动；左侧工具栏按「选择 / 绘制 / 连接 / 导出」分组，配图标、名称和快捷键；图形库标签（流程图 / 图形 / 电路 / 组件）紧凑排版，文字不再被挤压
- **几何变换**：任意角度旋转、任意比例缩放、水平/垂直翻转，基于 3×3 齐次坐标矩阵
- **撤销**：全量状态快照，支持 Ctrl+Z，撤销按钮状态实时同步
- **一键清屏**：清除所有图形（可撤销）
- **文件持久化**：`.vflow` JSON 格式保存/打开，支持继续编辑
- **导出**：全画布导出 PNG / SVG，矩形区域导出 PNG
- **自研算法**：Bresenham 直线、DDA、中点圆、中点椭圆、de Casteljau 任意阶 Bezier、Catmull-Rom 样条、扫描线填充（AET）、Cohen-Sutherland 裁剪、Wu 反走样、齐次坐标矩阵变换

## 说明

项目采用双渲染管线：

- **编辑画布**：使用 `CanvasRenderer` 在 tkinter Canvas 上做矢量渲染，保证缩放、平移、拖拽时的实时流畅交互。
- **PNG 导出与算法回放**：使用 `Renderer`，所有几何图元由自研像素级算法（Bresenham、中点圆/椭圆、扫描线填充、Bezier 等）逐像素生成后写入 Pillow 后缓冲区，再保存为 PNG。
- **SVG 导出**：使用 `SvgRenderer`，把图形、线条、连接线、文字和 Bezier 曲线转换为 SVG 元素，位图图片以内嵌 `<image>` 保存。

Pillow 只用于像素缓冲区、文本渲染和 PNG 保存。导出与回放路径上的核心几何图形没有使用 `ImageDraw.line`、`ImageDraw.rectangle`、`ImageDraw.ellipse` 等系统绘图函数，完整保留了图形学算法的展示价值。
