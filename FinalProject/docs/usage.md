# VectorFlow 使用说明

## 启动

```powershell
conda activate CGLab
python src/main.py
```

## 基本操作

| 操作 | 方法 |
|------|------|
| 放置流程图图元 | 左侧图元库选择类型，然后在画布点击 |
| 绘制直线 | 顶部工具栏选择"直线"，在画布拖拽 |
| 添加文本 | 选择"文本"工具，点击画布，直接在内联编辑框中输入（支持多行） |
| 编辑已有文本 | 双击文本图形，在内联编辑框中修改 |
| 连接图元 | 选择"连接线"工具，从一个图元拖拽到另一个图元（拖拽过程显示预览线） |
| 选择图形 | 选择工具下点击图形 |
| 框选多个图形 | 选择工具下从空白处拖拽矩形框 |
| 移动图形 | 选中后拖拽 |
| 编辑图元文本 | 双击流程图图元 |
| 区域导出 PNG | 选择"区域导出"工具，在画布拖拽选取矩形区域，保存为 PNG |
| 平移画布 | 鼠标中键拖拽 |
| 缩放画布 | Ctrl + 鼠标滚轮 |

## 右侧属性面板

### 图形样式
- 填充色、描边色、描边线宽
- 旋转角度（-360° ~ 360°，任意值）
- 缩放比例（10% ~ 500%，任意值）
- 水平/垂直翻转

### 连接线样式
- 线型：直线 / 折线 / 曲线
- 终点箭头：→ 实心 / ▷ 空心 / ◇ 菱形 / ● 圆点 / 无
- 起点箭头：同上
- 线条样式：实线 / 虚线 / 点线

### 文本样式
- 对齐方式：居中 / 左对齐 / 右对齐
- 加粗
- 文本颜色

| L | 直线工具 |
| T | 文本工具 |
| K | 连接线工具 |

## 内联文本编辑

- 点击画布或双击已有文本时，在画布上弹出内联编辑框
- **Ctrl+Enter**：确认提交
- **Escape**：取消编辑
- 支持多行输入，渲染时按行分割显示

## 复制粘贴

- 框选多个图形后 Ctrl+C / Ctrl+V 可复制粘贴
- 如果被复制的图形两端都在选区内，关联的连接线会一并复制

## 文件格式

`.vflow` 是 JSON 文件，保存画布设置、图形列表、连接线、样式、文本和变换状态。重新打开后可以继续编辑。

## 撤销/重做

Ctrl+Z / Ctrl+Y / Ctrl+Shift+Z 覆盖所有编辑操作：放置图元、绘制直线、连接线、文本、移动、复制粘贴、删除、旋转、缩放、翻转、样式修改、文本编辑、区域导出。

## 课程设计说明

程序使用 tkinter 负责窗口、菜单、按钮和事件处理；画布中的几何图形由自研算法生成像素后写入 Pillow Image。文本由 Pillow 字体渲染，不作为几何绘图算法的一部分。
